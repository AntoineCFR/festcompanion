# Images de fond (par festival)

Déposez ici les fonds d'écran, nommés par identifiant de festival :

- `<festivalId>_home.jpg`     → fond de la page **Accueil** (countdown)
- `<festivalId>_featured.jpg` → fond de la page **À la une**

Exemples :
- `1_home.jpg`, `1_featured.jpg`  → Extrema Outdoor (festival_id = 1)
- `2_home.jpg`, `2_featured.jpg`  → Awakenings (festival_id = 2)

Format conseillé : JPG portrait ~1080×1920, plutôt sombre/contrasté
(un voile dégradé est appliqué par-dessus pour la lisibilité).

Si une image est absente, l'app retombe proprement sur le fond uni du thème.
